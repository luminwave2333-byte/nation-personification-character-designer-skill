# Reference Deconstruction: 方法级拆解

## 1. 当前参考图的角色系统

从图面可见，角色使用：

- 黑色高位双长马尾与齐刘海
- 银色冠饰/传统头面联想式头部结构
- 双侧黄色流苏
- 白色短款现代翻领外套
- 红色贴身高领/立领内层
- 红黑龙形主纹样
- 纵向书法纹理
- 高腰短 A 字/百褶裙
- 腰部圆形身份挂牌与黄色长穗
- 单侧黑色长袜，另一腿留白
- 黑白双色中跟鞋
- 中式长剑与黄色剑穗

该组合的价值不在任何单件，而在“功能分工”。

## 2. 功能分工表

| 部位 | 视觉职责 | 文化职责 | 动态职责 |
|---|---|---|---|
| 双长马尾 | 拉高并拉长剪影 | 年轻化、舞台化 | 大幅摆动 |
| 冠饰 | 顶部身份锚点 | 礼仪/传统联想 | 低 |
| 黄流苏 | 点色 | 传统饰件联想 | 高 |
| 白短外套 | 现代权力骨架 | 国家/外交/礼服感 | 低 |
| 红内搭 | 脸胸焦点 | 红色文化联想 | 低 |
| 龙纹 | H1 主图腾 | 中华/权力/历史 | 中 |
| 书法 | H2 纹理 | 文字文化 | 裙摆运动时产生竖向流线 |
| 短裙 | 扩张下半身 | 现代二次元/舞台 | 高 |
| 单袜 | 破坏对称 | 无须承担文化事实 | 中 |
| 长剑 | 对角线 | 文武/武力 | 高 |
| 黑白鞋 | 收束脚部 | 表面纹样补充 | 低 |

## 3. 形状语言

主形：长竖线。

- 马尾
- 腿
- 书法
- 流苏

辅助形：S 线。

- 龙身
- 发尾
- 纹样

破局线：强对角线。

- 长剑

所以角色画面并非“很多中国元素”，而是：

> 竖线建立礼仪与威严，S 线建立文化生命感，剑的对角线建立危险。

## 4. 色彩不是四种颜色，而是四种职业

- 白：空间与高级感
- 黑：结构、文字、头发
- 红：攻击点
- 金黄：运动点

新角色可替换颜色，但要保留“颜色有职责”的思想。

## 5. 为什么裙子上大量文字仍能成立

因为裙片本身是扇形扩张结构，纵向文字把视觉拉回竖直方向。文字在这里部分承担“视觉织纹”功能。

风险：AI 视频中会出现笔画漂移和高频闪烁。

生产版应改成：

```text
主裙片 1–2 处大字/完整句
次裙片 墨痕、飞白、印章块
其余 留白
```

## 6. 同类国拟群像常见的有效方法

从相关国拟/五常作品的公开缩略图可以观察到一种常见的群像设计策略：

1. 先用国家色域或局部旗色建立快速识别。
2. 再用不同服装原型区分人物：礼服、现代时装、军装/夹克、长袍、舞台服。
3. 头发和头饰做强差异。
4. 每人配一个大体量道具或表演性物件。
5. 同框时使用轮廓和色块区别，而不是要求观众读文字。

高级化时应把第 1 步从“直接国旗色”升级成“色彩职责”，把第 4 步从“随便拿东西”升级成“道具与历史/现代证据连接”。

## 7. 群像轮廓占位建议

五人组可分：

```text
角色 A：A/X + 长发 + 对角线武器
角色 B：T/倒三角 + 短夹克 + 前冲线
角色 C：I/H + 长外套 + 垂直手杖
角色 D：雕塑 X/A + 弧形裙摆 + 扇形道具
角色 E：柱体/厚 A + 高头饰 + 重垂直道具
```

即使全转黑色，仍有差异。

## 8. 新角色不应复制的原稿组合

禁止在一个新角色上同时出现：

- 黑色高双马尾
- 银色冠饰
- 双黄流苏
- 白短西装
- 红高领内搭
- 满裙书法短裙
- 单侧黑色长袜
- 黑白花纹中跟鞋
- 长剑
- 龙纹主图腾

只要组合相似度过高，即使改国籍或换颜色，仍属于结构复制。
